# CustomHealthIndicator

A Minecraft Forge 1.16.5 client-side mod that shows enemy health in real time.

## Features

- **Colored nametags** — player nametags display current HP with a heart icon. Color shifts from green to red as HP drops, and pulses when critically low
- **Flying damage numbers** — when you hit someone, a damage number flies out from your crosshair in a random direction
- **Low HP alert** — a large red number appears below your crosshair when the target has 5 HP or less
- **Keybindings** — adjust number sizes on the fly without restarting:
  - `Shift + =` / `Shift + -` — increase/decrease flying number size
  - `Shift + Numpad+` / `Shift + Numpad-` — increase/decrease low HP number size

## Installation

1. Download `customhealthindicator-1.5-forge-1.16.5.jar` from [Releases](../../releases)
2. Place it in your `.minecraft/mods/` folder
3. Launch Minecraft with **Forge 1.16.5**

## Requirements

- Minecraft 1.16.5
- Forge 36.x
